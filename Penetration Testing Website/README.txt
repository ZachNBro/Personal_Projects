***Description***

An interactive pentetration testing scenerio focusing on access levels and proper
authentication and authorization mitigations in vulnerable applications.

***Directions***

To initiate this experience correctly, please double click the 'home.html' file first and
navigate from there.

***Requirements***

Any browser capable of viewing HTML, CSS and JavaScript. 

***Recommendations***

It is recommended to have the following publications/web sites available for reference:

-NIST 800-37
-NIST 800-53
-NIST 800-63B
-OWASP Cheat Sheet Series
